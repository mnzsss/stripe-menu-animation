- [Prettier](https://prettier.io)

## 🗂 Como clonar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/mnzsss/stripe-menu-animation.git

    # Entrar no diretório
    $ cd ../stripe-menu-animation

    # Instalar Dependências
    $ yarn

    # Inicar projeto
    $ yarn start

```
